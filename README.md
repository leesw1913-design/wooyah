1. 시작하기 — 설치 없이 바로 사용
이 제품은 HTML 파일 1개로 구성된 프로그램입니다. 별도 설치, 회원가입, 인터넷 연결이 전혀 필요 없습니다.
① 구매 시 받은 영업실적_대시보드.html 파일을 PC에 저장합니다 (바탕화면 추천).
② 파일을 더블클릭하면 크롬·엣지 등 브라우저에서 열립니다.
③ 처음에는 데이터가 없어 데이터 입력 안내 화면이 표시됩니다. 정상입니다.
TIP 권장 브라우저: Chrome 또는 Edge 최신 버전. 회사 보안 PC에서도 별도 권한 없이 동작합니다.
2. 데이터 입력 ① — 엑셀 업로드 (권장)
여러 명·여러 달 데이터를 한 번에 넣을 때 사용합니다.
① 왼쪽 메뉴 [실적 업로드] → [입력 양식 다운로드] 클릭
② 다운로드된 엑셀에 아래 규칙대로 실적을 입력하고 저장
③ 엑셀 파일을 업로드 영역에 드래그&드롭 (또는 클릭해서 선택)
④ 미리보기에서 내용 확인 후 [✓ 대시보드에 적용] 클릭
컬럼 필수 입력 예시 설명
구분1 필수 수도권사업부 팀/사업부 이름
구분2 선택 강남지사 지사/지점 이름
이름 권장 홍길동 영업사원 이름 (개인 화면 생성)
년도 권장 26 또는 2026 두 자리·네 자리 모두 인식
월 필수 6 1~12만 인식 (그 외 행은 제외)
건수 목표/실적 권장 3 / 2 계약 건수 (건)
매출 목표/실적 필수 20000000 원 단위 숫자 (콤마 없이)
이익 목표/실적 권장 6000000 원 단위 숫자
입력 규칙
TIP 같은 인원·같은 년월 데이터를 다시 올리면 기존 값이 새 값으로 갱신됩니다. 매달 누적 파일 전체를 올려도, 새 달
데이터만 올려도 됩니다. 미래 월의 목표만 미리 입력해 두는 것도 가능합니다.

3. 데이터 입력 ② — 간편 입력 (한 건씩)
외근 중이거나 한두 건만 고칠 때 엑셀 없이 바로 입력하는 기능입니다.
① 왼쪽 메뉴 [간편 입력] 클릭
② 소속·이름·년월을 입력 — 기존 인원은 이름만 쳐도 소속이 자동으로 채워집니다
③ 건수·매출·이익의 목표/실적을 입력하고 [✓ 대시보드에 반영]
TIP 이미 입력된 인원의 같은 년월을 선택하면 기존 값을 자동으로 불러와 수정 모드로 전환됩니다 (주황색 안내 표시).
4. 저장하기 — 반드시 읽어주세요
입력한 데이터를 파일에 영구 보관하려면 저장 버튼을 눌러야 합니다.
① 왼쪽 메뉴 [실적 업로드] 화면으로 이동
② [HTML 저장] 클릭 → 데이터가 포함된 새 파일이 다운로드 폴더에 저장됩니다
③ 다음부터는 새로 저장된 파일을 열어서 사용하세요 (파일명에 저장 날짜가 붙습니다)
주의 — 저장하지 않고 창을 닫으면 입력 내용이 사라집니다
브라우저 화면에서 입력한 데이터는 [HTML 저장]을 누르기 전까지는 화면에만 존재합니다. 입력 작업 후에는 꼭
HTML 저장을 눌러 새 파일을 받아 두세요. 날짜별 파일이 자동으로 백업본이 되는 장점도 있습니다.

6. 모바일에서 사용하기
같은 파일이 휴대폰에서 자동으로 모바일 화면으로 전환됩니다.
① 카카오톡 "나에게 보내기", 이메일 등으로 HTML 파일을 휴대폰에 전송
② 파일을 열 때 Chrome 또는 삼성 인터넷을 선택
③ 좌측 상단 메뉴(≡) 버튼으로 팀/지사/개인 이동, 간편 입력도 그대로 사용 가능
7. 직급 표시 설정 (선택 사항)
순위표에서 특정 인원에게 지사장·팀장 등의 직급을 표시할 수 있습니다. 간단한 텍스트 수정으로 가능합니다.
① HTML 파일을 마우스 오른쪽 클릭 → 연결 프로그램 → 메모장으로 열기
② Ctrl+F로 BRANCH_MGR 검색
③ 따옴표 안에 본인 조직의 이름을 입력 후 저장 — 예: new Set(['홍길동','김철수'])
BRANCH_MGR(지사장) · TEAM_MGR(팀장) · TOTAL_TEAM(총괄) · NEW_REP(신입) 네 가지 명단이
있으며, 비워 두면 전원 '영업사원'으로 표시됩니다.
8. 자주 묻는 질문 (FAQ)
질문 답변
엑셀을 올렸는데 일부 행이 빠졌어요 월 값이 1~12 범위 밖이거나 빈 행은 자동 제외됩니다. 적용 후 알림 문구에서 제외된
행 수를 확인할 수 있습니다.
연도를 2026으로 입력해도 되나요? 네. 26, 2026 어느 쪽으로 입력해도 자동으로 인식합니다.
내년 목표를 미리 넣을 수 있나요? 네. 미래 년월의 목표만 입력해 두면 기준월 선택기에 등록되고, 요약 화면에는
'예정'으로 표시됩니다.
저장한 파일이 어디 있나요? 브라우저의 다운로드 폴더에 "세일즈매니지먼트_대시보드_날짜.html" 형태로
저장됩니다.
파일이 메모장으로 열려요 파일 우클릭 → 연결 프로그램 → Chrome을 선택하세요.
데이터가 외부로 전송되나요? 아니요. 인터넷 연결 없이 동작하는 완전 오프라인 파일로, 데이터는 파일 안에만
저장됩니다.
엑셀 없이도 쓸 수 있나요? 네. 간편 입력만으로도 모든 데이터를 입력할 수 있습니다.
실수로 잘못 올렸어요 올바른 데이터로 다시 업로드하면 같은 인원·년월 기준으로 덮어쓰기됩니다. 저장
전이라면 파일을 닫고 마지막 저장본을 다시 열어도 됩니다.
